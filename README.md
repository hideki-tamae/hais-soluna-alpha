### Mathematical Model
$$Unlock(\lambda, \omega, t) = C(\lambda) \times R(\omega) \times D(t)$$

* **$C(\lambda)$ (Capacity)**: 逆境の深さ（$\lambda$）に基づく生涯発行キャパシティ。

* **$D(t)$ (Decay)**: 停滞減衰関数。持続的なケアの実践を促す。
